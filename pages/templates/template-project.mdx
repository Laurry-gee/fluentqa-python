# Python Templates Project

```shell
cookiecutter https://github.com/fluent-qa/fluent-pypro-template.git
```

Follow the instruction to generate a template project.

## Use Poetry as Dependency Managment tools

```shell
poetry init 
poetry add <package_name>
poetry add <package_name> -D
poetry build
poetry install
```

## How to Create template

- cookiecutter based 
- generate files: hooks
  * post_gen_project.py
  * pre_gen_project.py
